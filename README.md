💰 BudgetBuddy

BudgetBuddy is a personal finance tracker built using Python, Streamlit, and Supabase.

Features
- Secure user authentication
- Add and manage expenses
- Record income
- View transaction history
- Interactive financial reports
- Expense breakdown charts
- Monthly spending trends
- Profile management
- Password hashing using bcrypt

Technologies Used
- Python
- Streamlit
- Supabase
- Pandas
- Plotly
- bcrypt
